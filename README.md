📊 Agregador de Dados para Imposto de Renda (Excel)
📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio da plataforma DIO (Digital Innovation One), com o objetivo de aplicar na prática os conhecimentos adquiridos ao longo do curso.

A proposta consiste na criação de uma ferramenta em Microsoft Excel capaz de organizar e centralizar informações essenciais para a declaração de imposto de renda, proporcionando maior controle, praticidade e confiabilidade ao usuário.

🎯 Objetivo

Desenvolver um agregador de dados no Excel que permita:

Registrar entradas financeiras de forma organizada
Validar automaticamente os dados inseridos
Facilitar a navegação entre diferentes áreas da planilha
Oferecer uma experiência simples e eficiente
🛠️ Funcionalidades
📥 Registro de dados financeiros (receitas, despesas, etc.)
✅ Validação automática de dados
📑 Menus de navegação interativos
🔗 Links rápidos para acesso a seções importantes
📊 Organização clara das informações para apoio na declaração do IR
🧠 Conceitos Aplicados

Durante o desenvolvimento deste projeto, foram utilizados:

Estruturação de planilhas no Excel
Validação de dados
Criação de interfaces amigáveis
Organização lógica de informações
Documentação técnica
🚀 Como Utilizar
Faça o download do arquivo Excel deste repositório
Abra no Microsoft Excel
Navegue pelos menus disponíveis
Insira seus dados nos campos indicados
Utilize as validações e links para facilitar o uso
